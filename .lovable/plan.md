

# Feature Showroom Website

A professional, SaaS-style multipage website for sales teams to demo web features to clients. Neutral white/slate base with an indigo accent color.

---

## Global Layout & Navigation

- **Fixed top navbar** with logo ("Feature Showroom"), navigation links (Home, About), and a **"Features" dropdown** listing all 7 feature pages
- **Consistent page template** used across all feature pages: Hero → Demo Lab → Sales Cheat Sheet
- **Smooth page transitions** and hover animations using CSS transitions
- **Indigo/Electric Blue accent** color scheme on a clean white/slate background

---

## Home Page (/)

- Hero section introducing the showroom concept — "Explore battle-tested web features, ready to deploy"
- Grid of 7 feature cards linking to each feature page, with icons and short descriptions
- Clean, modern SaaS landing page feel

---

## Feature Pages (7 total)

Each page follows the same structure:

1. **Educational Hero** — Feature title, subtitle, and "Why it Matters" section with 3 ROI/engagement bullet points
2. **Demo Lab** — 2-3 live, functional variants of the feature
3. **Sales Cheat Sheet** — "Pro Tip" box explaining which client types benefit from which variant

### 1. Photo Gallery (`/photo-gallery`)
- **Variant A:** Masonry grid layout with lightbox overlay on click
- **Variant B:** Carousel/slider with thumbnail navigation strip

### 2. Testimonials (`/testimonials`)
- **Variant A:** Auto-scrolling horizontal marquee of quote cards with logos
- **Variant B:** 3-column grid with star ratings, avatars, and quotes

### 3. FAQs (`/faqs`)
- **Variant A:** Clean accordion with smooth expand/collapse animations
- **Variant B:** Category-based FAQ with sidebar filter to narrow topics

### 4. Blog (`/blog`)
- **Variant A:** Standard list view with featured images, dates, and read time estimates
- **Variant B:** Card-based grid layout with category tags and hover effects

### 5. Portfolio / Case Studies (`/portfolio`)
- **Variant A:** Results-driven layout with Challenge → Solution → Results metrics
- **Variant B:** Visual gallery grid with hover-reveal descriptions

### 6. Newsletter Sign-up (`/newsletter`)
- **Variant A:** Inline footer-style email capture form
- **Variant B:** Centered lead magnet section with image, value proposition, and sign-up form

### 7. Advanced Pop-ups (`/popups`)
- **Variant A:** "Trigger Demo" button that shows an exit-intent style modal overlay
- **Variant B:** "Trigger Demo" button that shows a slide-in promotional notification in the bottom corner

---

## Design & Interactions

- Professional SaaS aesthetic with generous whitespace
- Indigo accent color for CTAs, links, and highlights
- Smooth hover effects on cards and interactive elements
- Animated section entrances using fade-in and scale-in transitions
- All demo components use placeholder/mock data (no backend needed)
